# 📈 SmallBiz Pro - Financial Dashboard

A professional, enterprise-grade financial management dashboard built with **React** and **Firebase**. This tool allows small business owners to track sales, manage payroll, and analyze growth in real-time.

---

## ✨ Features
- **Real-time Analytics:** Interactive charts showing cash flow trends using Chart.js.
- **Financial Tracking:** Quickly log sales and expenses with categorized entries.
- **Staff & Payroll:** Manage employee records and track monthly salary commitments.
- **CSV Reporting:** Export your financial data to professional spreadsheets with one click.
- **Dark Mode:** Fully responsive theme that respects user preference.
- **Secure:** Database keys are protected via environment variables (`.env`).

## 🛠️ Tech Stack
- **Frontend:** React.js, Lucide Icons
- **Backend:** Firebase (Firestore & Authentication)
- **Charts:** Chart.js
- **Styling:** Custom CSS-in-JS (Modern, clean UI)
